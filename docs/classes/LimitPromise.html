Class LimitPromise

异步[并发]数的函数.

核心思想就是在并发数大于限制的时候，暂时先把操作放入通过promise组装一下，然后放入数组，每次执行完异步回调后就将操作按先入先出的顺序取出，然后执行，保证最大并发数在限制范围内。
