    • Defined in CMath3D/Euler.ts:38

    欧拉角描述一个旋转变换，通过指定轴顺序和其各个轴向上的指定旋转角度来旋转一个物体。

    Parameters

    • Optional x: number

      用弧度表示x轴旋转量。 默认值是 0

    • Optional y: number

      用弧度表示y轴旋转量。 默认值是 0

    • Optional z: number

      用弧度表示z轴旋转量。 默认值是 0

    • Optional order: string

      表示旋转顺序的字符串，默认为'XYZ'（必须是大写）

    Returns Euler

order

order: string = Euler.DefaultOrder
  • Defined in CMath3D/Euler.ts:30

order值应用于旋转顺序。默认值为 'XYZ'，这意味着对象将首先是 绕X轴旋转，然后是Y轴，最后是Z轴。其他可能性包括: 'YZX'， 'ZXY'， 'XZY'， 'YXZ'和'ZYX'。这些必须是大写字母。
