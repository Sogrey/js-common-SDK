TypeScript-SDK-Template

使用TypeScript开发SDK模板

  • tsdk.js 是 CommonJS 模块。这是一种被 NodeJS 使用的模块类型，看起来像 const myModule = require('my-module')
  • tsdk.module.js 是 ECMAScript 模块，由 ES6 定义，看起来类似 import MyModule from 'my-module'
  • tsdk.modern.js modern ES2017 output (see below)
  • tsdk.umd.js 是 UMD 模块
  • tsdk.d.ts 是 TypeScript 类型描述文件

In addition to the above formats, Microbundle also outputs a modern bundle specially designed to work in all modern browsers. This bundle preserves most modern JS features when compiling your code, but ensures the result runs in 95% of web browsers without needing to be transpiled.
